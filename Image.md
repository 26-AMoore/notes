# A 'picture' of an object
## Image attributes
- Type -- see types, real or virtual
- Size -- relative to object, bigger, smaller, same
- Orientation -- relative to object, erect or inverted

## Two types
Virtual
- Image that is formed when [[Rays]] are traced back to an imaginary point where it appears they all meet.
- In a mirror or lens
- Have negative image distance
