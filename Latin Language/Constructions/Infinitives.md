# Infinitive Forms
| Form            | PP  | Ends in | Translates as         |
| --------------- | --- | ------- | --------------------- |
| Present Active  | 2nd | *re*    | To [verb]             |
| Perfect Active  | 3rd | *sse*   | To have [verb]ed      |
| Present Passive | 2nd | *ri*    | To be [verb]ed        |
| Perfect Passive | 4th | *esse*  | To have been [verb]ed |
