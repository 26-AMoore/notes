# A surface which [[Reflection|Reflects]] [[Light]]
## Two main types
Concave
- Real or virtual images
Convex
- Only virtual images

### Also!
When any [[Rays|Ray]] parallel to the PA hits a concave mirror it will pass through the focal point

### For ray tracing draw three rays
- Parallel from object
- From object through F and into parallel
- Through C
